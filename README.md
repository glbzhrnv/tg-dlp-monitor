# TG-DLP Monitor

TG-DLP Monitor — прототип программного средства для выявления утечек данных и информации об актуальных угрозах организации в мессенджере Telegram.

Система выполняет сбор сообщений из заданных Telegram-источников, анализирует текстовые сообщения и графические вложения, выявляет признаки утечек конфиденциальной информации и формирует алерты для администратора.

## Основные возможности

- сбор сообщений из Telegram-каналов и групп;
- анализ текстовых сообщений;
- OCR-обработка изображений;
- выявление признаков утечек с помощью регулярных выражений;
- использование модели CatBoost для итоговой оценки риска;
- поддержка RuBERT-модели для семантического анализа текста;
- сохранение результатов анализа в базе данных;
- веб-интерфейс администратора для просмотра источников, сообщений и алертов.

## Стек технологий

- Python;
- FastAPI;
- Telethon;
- SQLite;
- CatBoost;
- RuBERT;
- EasyOCR;
- Jinja2;
- HTML/CSS.
